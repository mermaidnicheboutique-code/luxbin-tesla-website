## üêõ Troubleshooting

**Animations not working:**
- Check JavaScript console (F12)
- Ensure `script.js` is in same folder as `index.html`

**Fonts look wrong:**
- Check internet connection (fonts load from Google Fonts)
- Fallback to sans-serif if Google Fonts unavailable

**Mobile menu not appearing:**
- Resize browser to <768px width
- Hamburger menu should appear top-right

**Stars not moving:**
- Check CSS animations in `styles.css`
- Try hard refresh (Cmd+Shift+R on Mac)
